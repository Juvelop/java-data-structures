#### 클래스 선언 기본
- public 클래스 : 다른 패키지에서 사용할 수 있는 공개 클래스
- final 클래스 : 서브 클래스를 가질 수 없다.
- 파생 class : extends A
- 추상 클래스 : abstract 를 붙여 선언한 클래스로, 인스턴스를 만들 수 없다.
